# v0.4.1 Reference Clean Generation Audit

v0.4.1 is a generator-quality snapshot for the reference-derived clean receipt templates.

It does not replace the frozen `v0.2.0` benchmark payload or the `v0.4.0` Clean/Noisy LM Studio leaderboard. It documents that the current reference-style receipt generator passes item-line integrity checks before any future larger dataset expansion.

## Highlights

- Audited `23` clean synthetic receipt templates.
- Checked `124` item/service semantic lines.
- Confirmed normal item/service amount-missing errors: `0`.
- Confirmed invalid 0-yen item errors: `0`.
- Confirmed name + quantity + amount duplicate triples: `0`.
- Confirmed same-receipt normalized-name duplicates: `0`.
