## Safety

The report contains fictional synthetic receipts only. Real reference images are not copied into the report, README, or payload. The real references are used only as design and layout inspiration during local development.

## QA Note

`v0.4.2` is a receipt-structure library. It keeps all accepted structures, including several same-family variants that still share a few item-name/quantity/amount triples. Those overlaps are not published as final benchmark records; they are explicitly deferred to the next randomized 100-type dataset generation pass.
