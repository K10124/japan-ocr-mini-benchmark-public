# Legacy Initial 5-Receipt Sample

This folder preserves the earlier 5-receipt mini sample materials from the first public experiments.

These files are kept for historical comparison only. They are not the current benchmark payload.

For current OCR/VLM evaluation, use:

```text
release_v0.2.0/data/v0.2.0
```

The current v0.2.0 payload includes:

- `manifest.jsonl`
- `source_json/`
- `metadata/`
- `degradation_metadata/`
- `images_clean/`
- `images_noisy/`

Legacy folder contents:

- `eval/`: earlier comparison scripts and outputs
- `ground_truth/`: earlier ground-truth JSON files
- `images/`: earlier clean/noisy sample images
- `model_outputs/`: earlier model output examples
- `notes/`: earlier experiment notes
